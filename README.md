# Scoreboard

Tournament scorekeeper app for Mahjong.

Built with Solid.js and [TanStack Start](https://tanstack.com/start/latest).

## Development

We use some [Bun.js](https://bun.com/) features.

```sh
# set up
bun install
cp example.env .env   # edit accordingly
bun dev

# production build
bun run build
bun run start
```
